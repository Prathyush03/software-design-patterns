# software-design-patterns

This repo contains interactive html files for each design pattern

## Contents

- `ch*.html` — **Head First Design Patterns** chapter walkthroughs (Strategy, Observer, Decorator, Factory, Command, Adapter/Facade, Template Method, Iterator/Composite, State, Proxy, Compound patterns). Each is a self-contained step-by-step slideshow: naive attempt → failure → design principles → the pattern.
- `coding-patterns/` — **Grokking coding interview patterns** (Two Pointers, Sliding Window, DP, ...) in the same interactive format. Start at `coding-patterns/patterns-map.html`.
- `ood-case-studies/` — **Grokking the Object Oriented Design Interview** case studies (Parking Lot, ATM, Chess, ...) solved with the same Head First methodology: a naive God-object attempt collapses, the 5-step OOD recipe rebuilds it, and the design patterns emerge where the requirements demand them. Start at `ood-case-studies/index.html`.

Open any file directly in a browser — no build step, no dependencies.
